# bsvz

`bsvz` is a BSV foundation library for Zig.

Initial focus:

- primitives
- crypto
- script
- transaction
- SPV
- broadcast

Non-goals for the first milestone:

- full node functionality
- full wallet product APIs
- broad application-layer protocol coverage

## Zig Version

- Zig `0.15.2`

## Module Layout

- `bsvz.primitives`
- `bsvz.crypto`
- `bsvz.script`
- `bsvz.transaction`
- `bsvz.spv`
- `bsvz.broadcast`
- `bsvz.compat`
- `bsvz.wallet_core`

## Status

This repository is a brand-new standalone BSV library. The current state is an initial library-first skeleton.
